# 🏥 Healthcare Appointment Management System (MERN Stack)

A full-stack role-based healthcare platform that allows patients to book appointments with doctors, doctors to manage schedules, and admins to manage the entire system.

## 🚀 Features
### 👨‍⚕️ Patient
Register / Login securely
Browse doctors by specialization
Search doctors
Book appointments
View appointment status (Pending / Approved / Completed / Cancelled)
Cancel appointments
### 🩺 Doctor
Doctor dashboard
View assigned appointments
Approve / Reject / Complete appointments
View patient details
Real-time appointment updates
### 🛠️ Admin
Admin dashboard
Add new doctors
Upload doctor profile image
Manage users (patients/doctors/admins)
View all appointments
System control panel
### 🔐 Authentication & Security
JWT Authentication
Role-based access control (RBAC)
Password hashing with bcrypt
Protected routes for each role
Secure API middleware
### 📩 Notifications (Advanced)
SMS notifications using Twilio (appointment updates)
Email notifications (optional setup)
Instant alerts when doctor updates appointment status

## 🧱 Tech Stack
### Frontend
React.js
React Router DOM
Tailwind CSS
Axios
### Backend
Node.js
Express.js
MongoDB + Mongoose
### Services
JWT (Authentication)
Twilio (SMS notifications)
Nodemailer (Email notifications)
